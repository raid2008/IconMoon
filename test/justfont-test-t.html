GUI 設計師裝逼指南

清明掃墓回來路上又長又無聊，看到知乎上的問題「在你熟知的領域，如果由你撰寫一份《裝逼指南》，內容會是什麽？」就用爪機答了一下，大家笑笑就好，不要當真，哈哈哈：

  1. 用的軟件：現在不用 PS 了，直接上 Sketch.app，CD、Fireworks 之類的聽也沒聽過
  2. 每次提案之前都要先扯一下 Flat Design，但完全鄙視擬物風格也不好，要帶有自己的觀點，不對，不能叫擬物，應該是 skeuomorphism，字雖然長但發音絕對要清楚
  3. 哪怕幹的活性質再死美工也要放一點 Design Process，每個圖標都要有手繪稿，必須圖標設計完之後按照那個樣子好好重繪一遍，哪怕花的時間比圖標本身還長也值得
  4. 效果圖直接平鋪直敘給客戶或者放網上太土鼈了，套個 iPhone 的模版算湊活，最好是界面存手機裏，用個大光圈的照相機拍一下，DoF 的感覺一定不能少，而且要去咖啡館拍，跑星巴克你就輸了，怎麽說焦外也要有點花花草草明信片書架什麽的，我們管這個叫「情境化」，有條件的再找個嫩模貢獻一下身段出現在照片裏冒充使用者。
  5. 個人網站已經不流行放 deviantArt 鏈接了，要放就放那個粉色的籃球，什麽？ Dribbble 上只有 prospect 帳號？趕緊的，1500 rmb 一個買起來，也不貴。最好 Github 也放一個
  6. C/S B/S 架構啥意思？我只給 iOS 和 Mac 平台做過設計，不過最近在研究 Windows 8 的設計規範
  7. 切圖直接上 @2x ，心情好了給你個普通大小的。ldpi、mdpi、hdpi、xhdpi 這種是我們那兒新來的設計師鼓搗的，我不管 Android 這種破平台
  8. 私活這種三天兩頭有人找我做，不過一般懶得接〜怎麽收錢？對不起工行招行建行支付寶一律沒有，只收 Paypal，是美國那家，不是貝寶哦親
  9. 收費標准咱按時薪算的，改多少遍隨你，做完用 Ballpark 發你張 invoice 直接打錢就行
  10. 網頁咱也做，絕壁給你一個 responsive 的，2560 到 800 再到 iPhone iPad 每個分辨率都能適配，但是不好意思，IE6 到 IE9 不管，我們是有原則的，加錢也不做
  11. 電腦方面女設計師只能用 MacBook Air，屏雖然差點，但放包裏正好，男的必須 rMBP，跟人介紹的時候 Retina 要念准，「t 」的發音不能重了，別人問起你要說其實電視廣告裏那個也不標准，幫他糾正一下
  12. 別人聊起設計大師啥的，千萬別說你喜歡原研哉啊 Dieter Rams 啊，那都爛大街了，要挑小衆的說，可以表現一下愛國情懷，說你認識祖國第一代界面設計師 Robin.Z，昨天還在群裏跟他聊了一會兒天
